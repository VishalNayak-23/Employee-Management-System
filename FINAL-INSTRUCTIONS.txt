## 🚀 HOW TO USE THIS ZIP FILE

### Step 1: Extract the ZIP
1. Go to your Desktop
2. Find `employee-management-system.zip`
3. Right-click → Extract All
4. Choose destination folder
5. Click Extract

### Step 2: Open in VS Code
1. Open VS Code
2. Click: File → Open Folder
3. Navigate to extracted folder: `employee-management-system`
4. Click "Select Folder"

### Step 3: Open Terminal in VS Code
- Press: `Ctrl + ` (backtick key)
- Or: Menu → Terminal → New Terminal

### Step 4: Install & Run

**In Terminal, run these commands ONE BY ONE:**

```bash
# Install Backend
cd backend
npm install
node utils/seed.js
npm start
```

**Open NEW Terminal (Click + icon or Ctrl+Shift+`):**

```bash
# Install Frontend
cd frontend
npm install
npm run dev
```

### Step 5: Open Browser
- Go to: http://localhost:3000
