## 🆘 Need Help?

### Quick Fixes:

**MongoDB not starting?**
```bash
# Windows
net start MongoDB

# Mac/Linux
sudo systemctl start mongod
```

**Port already in use?**
```bash
npx kill-port 5000
npx kill-port 3000
```

**Module not found?**
```bash
cd backend
npm install

cd ../frontend
npm install
```
